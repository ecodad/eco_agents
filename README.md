# eco_agents
Eco's Shared Agents

This repository is for sharing agents that I have built.

AGENTS:

**Briefing Agent:** Daily and Weekly Briefs including upcomming calendar events, related action items and advisories. Designed for Anthropic cloud routines, multiple Google Calendars through Connectors, Discord push notification to smartphone with context pulled from GitHub saved Obsidian Vault using Tasks community plugin for action items.
