Eco's Shared Agents

This repository is for sharing agents that I have built.

AGENTS:

##Briefing Assistant##
The Briefing Assistant is an agentic workflow designed to streamline daily & weekly planning by bridging local knowledge management with cloud-based AI reasoning. Operating entirely in the cloud using a Pro subscription, the assistant securely accesses a GitHub repository containing an Obsidian vault to parse active markdown tasks managed by the Tasks community plugin.

Every morning, it synthesizes these active tasks and vault notes with your daily calendar agendas to identify immediate priorities and scheduling conflicts. The final output is delivered directly to your phone via a Discord (or Slack) push notification, while a permanent archive of the daily briefing is automatically written back to your Obsidian vault.
